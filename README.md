# 6-DOF Industrial Robotic Arm

## Overview
This repository contains the hardware design, simulation files, and documentation for a 6-DOF (Degrees of Freedom) industrial robotic arm. This system was developed by team **Neural Nexus** for the EN2160 Electronic Design Realization module. 

The project encompasses custom high-speed PCB designs, mechanical part models, and a complete simulation environment for testing kinematics and control logic.

## Repository Structure

* **`/Encoder PCB files`**
  * Contains the Altium Designer schematics, PCB layouts, and manufacturing files (Gerbers/NC Drill) for the joint encoder boards.
* **`/Main PCB files`**
  * Contains the Altium Designer project for the main control board, including power distribution and signal routing.
* **`/Part files`**
  * Mechanical CAD models (e.g., STEP/STL files) for the robotic arm links, base, and actuator mounts.
* **`/Robotic ARM Simulation`**
  * Simulation environment files to test and visualize the 6-DOF movements and control algorithms.

## Tools & Technologies Used
* **Hardware Design:** Altium Designer (Multi-layer PCB routing)
* **Simulation:** Made using Python
* **Mechanical CAD:** SolidWorks

## Getting Started
